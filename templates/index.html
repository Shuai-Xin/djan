帕雷托最適

Pareto Optimality

帕雷托最適是指資源分配上的一種最理想的狀態。此時你要增加一個人的福利，那必定要建立在另一個人的損失上。

若還沒到達最優，那就能進行帕雷托改進 ( Pareto improvment )，此時可以在沒人損失的狀態下使人變得更好。 可是在古典功利主義 ( utilitarianism，又譯作效益主義，此思想創始人為邊沁，耳熟吼，高中公民 ) 的眼中，資源指的是「幸福」，因此算是道德嚴格版的帕雷托最優。 舉例：若存在一個系統內只有有一個富翁跟一個乞丐，此時你要讓乞丐活下去就一定要從富翁那邊拿錢給他用，會造成富翁的損失，因此已經是帕雷托最優了！這個系統已經達到最佳的資源分配。但在邊沁的眼中，富翁少一點錢不會造成多少幸福感下降，但乞丐因此活了下去，是大大的幸福感上升，因此富翁應該給乞丐錢。 ( 眼熟吼，也可以翻譯成柏拉圖啦。順提，帕雷托是個義大利的經濟學家，知名的20/80定律也是他提出的 )
